Tetris

This project focuses on implementing a dynamic version of the Tetris game, incorporating novel features such as pieces with random shapes and sizes, additional movements like flipping and scaling, and a board that changes based on certain moves. The primary challenge involves managing these elements through dynamic arrays. Key tasks include coding in java, adhering to javaDoc standards for documentation, and ensuring the code compiles and runs as specified.


Video Conferencing WebApp

Pixochat is a video conferencing platform that emphasizes real-time engagement and is designed to foster meaningful human connections. The website offers a simple, user-friendly interface for hosting and joining meetings, with functionalities such as naming the meeting and restricting host controls.



Chrome Extension

This project involves creating a Chrome extension that customizes the new tab page. It features a randomized background image, the current time, a greeting that allows users to set their name, and a menu with additional functionalities like a random quote, Pomodoro timer, or sticky notes.
